使い方
  1. Power Automate または PAD ソリューションファイル（ZIP）をアップロード
  2. 解析開始ボタンをクリック（自動判定）
  3. 「AI要約」タブを選択し、「AI生成」ボタンをクリック
  4. 生成された内容を確認し、「CSV出力」ボタンでExcel形式で保存
対応形式:
✓ Power Automate (クラウドフロー)
✓ Power Automate Desktop (デスクトップフロー)
